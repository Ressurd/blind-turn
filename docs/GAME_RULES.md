# BLIND TURN 전투 규칙 V2

## 기본값

- 2~6명이 참여합니다.
- 기본 HP는 30, 수호자는 35입니다.
- 엔진은 2.5 피해를 정수로 계산하기 위해 `HP_SCALE = 2`를 사용합니다.
- 생존자가 1명이면 승리, 0명이면 무승부입니다.
- 속도, 숨김 동률 주사위, 행동 순서, 시선과 허점 공격은 V2에 존재하지 않습니다.

## 캐릭터

| 캐릭터 | 지속 효과 |
| --- | --- |
| 결투가 | 모든 합 최종값 +1 |
| 광전사 | 직접 공격 카드 피해 +1 |
| 수호자 | 최대/시작 HP 35 |
| 전술가 | 시작 카드 6장을 보고 5장 선택, 남은 1장은 덱에 무작위 삽입 |

## 덱과 손패

기본 덱은 속공×2, 강공×2, 방어×2, 회피×2, 반격×2의 10장입니다. 게임 시작 시 각자 섞어서 5장을 받습니다. 이후 라운드 시작마다 손패가 5장 미만이면 1장만 뽑고, 이미 5장이면 드로우를 생략합니다. 뽑기 더미가 비면 무덤(discardPile)만 섞어 새 뽑기 더미로 사용합니다. 영구 제거 카드는 재셔플되지 않습니다.

한 라운드에 손패에서 0~3장을 고릅니다. 카드를 고르면 선택 순서대로 첫 번째 빈 행동에 자동 예약되며, 필요할 때만 순서 편집창에서 위/아래로 이동하거나 취소합니다. 취소 뒤에는 뒤쪽 카드가 자동으로 당겨져 예약 순서가 항상 0부터 연속됩니다. 카드별 대상이나 추가 선택도 함께 저장합니다. 60초 안에 확정하지 않으면 서버가 카드 0장으로 확정합니다. 판정 전에는 상대에게 카드 수, 카드 종류, 순서와 대상이 공개되지 않습니다. 전원이 확정되면 먼저 각 플레이어가 사용한 카드 장수만 공개합니다.

## 라운드와 단계

예약한 첫 번째 카드끼리 1단계, 두 번째 카드끼리 2단계, 세 번째 카드끼리 3단계를 구성합니다. 같은 단계는 동시에 판정합니다.

- 단계 시작 당시 살아 있으면 그 단계 행동은 실행됩니다.
- 같은 단계에서 피해를 받아 사망해도 자신의 그 단계 행동은 취소되지 않습니다.
- 사망자의 다음 단계 카드와 사망한 대상을 향한 다음 단계 카드는 취소됩니다.
- 실행한 카드는 라운드 종료 시 버림 더미로 갑니다. 실행 전에 취소된 카드는 손패에 남습니다.

## 기본 카드

### 속공

- 피해 3
- 합 보너스 +2
- 회피 난이도 8

### 강공

- 피해 5
- 합 보너스 +0
- 회피 난이도 5

### 방어

그 단계에 받는 공격 계열 총 피해를 5 줄입니다. 피해 합계가 감소량보다 작으면 0이 됩니다. 반격/자해 피해에는 적용되지 않습니다.

### 회피

그 단계에 들어오는 공격마다 좌석 순서로 회피 주사위(1~10)를 굴립니다. `주사위 + 보너스 >= 공격 카드의 회피 난이도`이면 해당 공격 피해가 0입니다. 첫 실패 공격은 원래 피해 대신 고정 10 피해가 되고 회피가 종료되며, 같은 단계의 나머지 공격은 정상 피해를 줍니다.

### 반격

지정한 플레이어가 같은 단계에 반격 사용자를 공격하면 원래 공격을 취소하고 공격자에게 10, 반격 사용자에게 2.5 피해를 동시에 줍니다. 지정하지 않은 공격에는 발동하지 않습니다.

## 합

같은 단계에 두 플레이어가 서로 공격하면 합을 합니다.

```text
합 최종값 = d10 + 카드 합 보너스 + 결투가 보너스(해당 시 +1)
```

클라이언트는 서버가 확정한 값을 좌석 순서대로 `첫 번째 주사위 룰렛 → 첫 번째 결과 → 두 번째 주사위 룰렛 → 두 번째 결과 → 보정값 → 최종 비교 → 승자` 순서로 재생합니다. 동률이면 무승부와 재대결 회차를 표시한 뒤 같은 순서를 반복합니다. 룰렛은 결과를 바꾸지 않는 시각 연출입니다.

동률이면 두 플레이어가 모두 다시 굴립니다. 승자의 공격만 패자에게 적용됩니다. 광전사의 직접 공격 +1과 카드별 특수 효과는 승리한 공격 피해에 포함됩니다.

## 직업/공용 카드

각 직업은 4장, 공용 풀은 6장입니다. 모든 수치와 설명은 `packages/game-engine/src/cards/card-catalog.ts`의 데이터 정의가 단일 원본입니다.

- 결투가: 섬광 찌르기, 결투의 칼날, 우세한 연격, 완벽한 응수
- 광전사: 분쇄, 처형, 무모한 강타, 방어 파쇄
- 수호자: 요새, 중첩 방어, 응급 회복, 최후의 버팀
- 전술가: 회수, 전술 교체, 선별, 예측 방어
- 공용: 응급 처치, 집중 공격, 강화 방어, 연막 회피, 도박, 신속 보충

카드는 일반 공격보다 높은/낮은 피해, 합 보너스, 방어 관통, 매 공격 감소, 회복, 드로우, 버림 회수, 손패 교체, 최후 생존 등의 효과를 데이터로 조합합니다.

## 보상과 덱 성장

3, 6, 9… 라운드 판정 후 생존자는 자신의 직업 카드 2장과 공용 카드 1장, 총 3개의 서로 다른 후보를 받습니다. 이미 2장 보유한 카드는 후보에서 제외됩니다. 60초 안에 정확히 2장을 선택해 확정해야 하며, 시간 초과 시 서버가 서로 다른 유효 후보 2장을 선택합니다.

- 3라운드: 10장 + 2장 = 12장
- 6라운드: 12장 + 2장 = 14장
- 9라운드: 임시 16장에서 기존 카드 1장 영구 제거 = 15장
- 12라운드 이후: 임시 17장에서 기존 카드 2장 영구 제거 = 15장

같은 성장에서 획득한 새 카드 2장은 제거 후보가 아닙니다. 필요한 제거 수는 `현재 사용 가능 덱 - 15`로 계산합니다. 제거 카드는 별도 영구 제거 목록으로 이동하고 무덤이나 재셔플에 포함되지 않습니다. 같은 카드 ID는 최대 2장이고, 제거 후에도 공격 카드가 최소 1장 있어야 합니다. 제거 제한 시간도 60초이며 시간 초과 시 서버가 유효한 기존 카드를 자동 제거합니다.

## 채팅

- 같은 방 플레이어에게만 전송
- 서버 세션의 playerId/닉네임 사용
- 앞뒤 공백 제거, 연속 공백 정규화, 1~100자
- 플레이어당 초당 최대 2개
- 최근 50개를 보관하고 재접속 시 복원
- 사망자는 새 메시지를 보낼 수 없음
- 입장/퇴장/게임 시작 같은 시스템 메시지는 플레이어 메시지와 구분
