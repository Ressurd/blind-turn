# BLIND TURN 프로덕션 플레이테스트 체크리스트

## 배포와 보안

- [ ] Web `/api/health`와 Server `/health`가 200
- [ ] health 응답에 방 코드, 닉네임, 토큰, 손패/선택 행동이 없음
- [ ] 허용한 Web origin만 Socket.IO 연결 가능
- [ ] Server 단일 인스턴스와 재시작 시 방 소멸 정책 확인

## 턴 선택

- [ ] 모든 직업 시작 덱 10장/손패 5장
- [ ] 전술가 시작 손패 선택 UI가 없음
- [ ] 한 턴에 카드 한 장만 선택되고 새 선택이 이전 선택을 교체
- [ ] 선택 취소/PASS/확정 동작
- [ ] 60초 미확정은 PASS, 카드 소비 없음
- [ ] 확정 전 상대 행동 카드/대상/추가 선택이 없음
- [ ] 상대 카드에 손패/덱/무덤/전체 장수와 확정 여부만 표시

## 판정과 재생

- [ ] 일반 공격
- [ ] 방어 중 피격
- [ ] 회피 연속 성공
- [ ] 회피 실패 고정 10 피해
- [ ] 반격 동시 피해
- [ ] 합 동률 재굴림과 승자 피해
- [ ] 합 패배 뒤 다음 행동 취소 규칙/문구가 없음
- [ ] 같은 턴 사망자의 확정 행동도 실행
- [ ] HP는 DAMAGE_APPLIED 재생 전 먼저 감소하지 않음
- [ ] 건너뛰기 후 displayState가 serverState와 일치
- [ ] 연출 중 다음 턴 이벤트가 와도 재생 큐 순서 유지
- [ ] 모션 감소 설정에서 핵심 결과 표시 유지

## 성장

- [ ] 3·6·9…턴 종료 후 성장, 게임 종료 후에는 없음
- [ ] 일반 직업 후보 3장 = 직업 2 + 공용 1
- [ ] 전술가 후보 4장 = 전술가 3 + 공용 1
- [ ] 후보 ID 중복 없음, 보유 제한 카드 제외
- [ ] 모든 직업 정확히 2장만 확정
- [ ] 60초 후 서버 RandomSource로 서로 다른 2장 자동 선택
- [ ] 후보/선택/포기 카드는 다른 플레이어에게 비공개
- [ ] 전술가도 2장만 추가되고 최대 덱 15장 유지

## 재접속

- [ ] 손패/선택 행동/대상/확정/기한 복원
- [ ] 전술가의 동일한 후보 4장과 현재 선택 복원, 재생성 없음
- [ ] 확정한 성장 선택 변경 불가
- [ ] 덱 제거 선택/기한 복원
- [ ] 전투 재생 도중 새로고침 시 최종 HP 선반영 없음

## 화면

- [ ] 1366×768과 1920×1080 데스크톱 전체 페이지 스크롤 없음
- [ ] 2인/6인 공격자·대상·카드 식별
- [ ] 6인 선택 카드 동시 공개
- [ ] 모바일 390px에서 방향선/피해 숫자/손패/행동 버튼이 겹치지 않음
- [ ] 전술가 성장 4장이 PC 4열 또는 2×2, 모바일 가로 스크롤
