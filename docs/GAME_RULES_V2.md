# BLIND TURN 전투 규칙 V2

## 기본값

- 2~6명이 참여합니다.
- 기본 HP는 30, 수호자는 35입니다.
- 엔진은 2.5 피해를 정수로 계산하기 위해 `HP_SCALE = 2`를 사용합니다.
- 생존자가 1명이면 승리, 0명이면 무승부입니다.
- 한 턴에 각 플레이어는 카드 1장 또는 `PASS`만 확정합니다.
- 속도, 숨김 동률 주사위, 행동 순서, 다단계 예약은 없습니다.

## 캐릭터

| 캐릭터 | 지속 효과 |
| --- | --- |
| 결투가 | 모든 합 최종값 +1 |
| 광전사 | 직접 공격 카드 피해 +1 |
| 수호자 | 최대/시작 HP 35 |
| 전술가 | 성장 때 전술가 카드 3장과 공용 카드 1장 중 2장 선택 |

전술가도 다른 직업과 똑같이 10장 덱을 섞고 시작 손패 5장을 받습니다. 시작 카드를 6장 확인하거나 5장을 다시 고르는 패시브는 없습니다.

## 덱과 손패

기본 덱은 속공×2, 강공×2, 방어×2, 회피×2, 반격×2의 10장입니다. 게임 시작 시 5장을 받고, 다음 턴 시작마다 손패가 5장 미만이면 1장만 뽑습니다. 뽑기 더미가 비면 무덤만 섞어 새 뽑기 더미로 사용합니다. 영구 제거 카드는 재셔플되지 않습니다.

## 턴 선택

행동 제한 시간은 60초입니다.

1. 플레이어 한 명을 대상으로 고릅니다. 자신을 고르면 방어·회피·유틸리티처럼 자신/대상 없음 카드를 볼 수 있습니다.
2. 손패에서 카드 한 장을 고르거나 `PASS`를 선택합니다.
3. 확정 전에는 카드를 바꾸거나 선택을 지울 수 있습니다.
4. `PASS`는 카드를 소비하지 않습니다.
5. 제한 시간까지 확정하지 않은 플레이어는 서버가 `PASS`로 확정합니다.

확정 전 다른 플레이어에게는 행동 카드, 대상, 추가 선택이 공개되지 않습니다. 공개되는 정보는 손패/덱/무덤/전체 덱 장수와 행동 확정 여부뿐입니다. 전원 확정 후에만 선택 카드를 동시에 공개합니다.

## 동시 판정

전원이 확정하면 한 턴의 행동을 다음 순서로 판정합니다.

1. 모든 선택 카드와 대상을 동시에 공개
2. 상호 공격의 합, 지정 반격, 회피, 방어, 유틸리티 효과 계산
3. 회복과 모든 피해를 턴 시작 HP 기준으로 모음
4. HP를 동시에 갱신
5. 사망 처리
6. 사용한 카드 버림 처리
7. 3의 배수 턴이면 성장, 아니면 다음 턴 시작

이번 턴 시작 시 살아서 확정한 행동은 같은 턴에 치명 피해를 받아도 실행됩니다. 사용하기로 확정한 카드는 효과가 발동하지 않았더라도 버림 더미로 갑니다. `PASS`만 카드를 소비하지 않습니다.

## 기본 카드

- 속공: 피해 3, 합 +2, 회피 난이도 8
- 강공: 피해 5, 합 +0, 회피 난이도 5
- 방어: 이번 턴 공격 계열 총 피해를 5 감소. 반격/자해에는 미적용
- 회피: `d10 + 보너스 >= 회피 난이도`면 피해 없음. 첫 실패는 고정 10 피해
- 반격: 지정 플레이어가 자신을 공격하면 원래 공격을 취소하고 공격자 10, 자신 2.5 피해

## 합

두 플레이어가 서로를 공격하면 합을 한 번 처리합니다.

```text
합 최종값 = d10 + 카드 합 보너스 + 결투가 보너스(해당 시 +1)
```

동률이면 양쪽이 다시 굴립니다. 승자의 현재 공격만 패자에게 적용되고 양쪽이 사용한 카드는 버립니다. 합 패배 때문에 별도의 다음 행동을 취소하는 규칙은 없습니다.

## 성장과 전술가

게임이 끝나지 않았다면 3, 6, 9, 12…턴 판정 후 생존자가 성장 카드를 고릅니다.

- 일반 직업: 자신의 직업 카드 2장 + 공용 카드 1장, 총 3장 중 정확히 2장
- 전술가: 전술가 카드 3장 + 공용 카드 1장, 총 4장 중 정확히 2장

후보 ID는 서로 달라야 하고 이미 카드별 최대 보유 수량 2장에 도달한 카드는 제외합니다. 전술가의 유효 직업 카드가 3장보다 적으면 다른 유효 전술가 카드를 먼저 찾고, 그래도 부족하면 중복되지 않는 공용 카드로 채웁니다. 서로 다른 후보를 만들 수 없으면 서버가 명확한 오류로 안전하게 중단합니다.

선택 제한 시간은 60초입니다. 시간 초과 시 서버 `RandomSource`가 후보 중 서로 다른 2장을 자동 선택합니다. 전술가도 획득하는 카드는 정확히 2장이고 최대 덱 크기는 15장으로 같습니다.

- 3턴: 10장 → 12장
- 6턴: 12장 → 14장
- 9턴: 임시 16장 → 기존 카드 1장 제거 → 15장
- 12턴 이후: 임시 17장 → 기존 카드 2장 제거 → 15장

같은 성장에서 받은 새 카드 2장은 즉시 제거할 수 없습니다. 제거 후에도 공격 카드는 최소 1장 남아야 합니다.

## 비공개 정보와 재접속

다른 플레이어는 성장 선택 중인지와 완료 여부만 알 수 있습니다. 후보, 선택한 카드, 포기한 카드와 순서는 본인에게만 보입니다.

재접속하면 서버가 다음 정보를 그대로 복원합니다.

- 손패와 이번 턴 선택 행동/대상/추가 선택/확정 여부
- 성장 후보 3장 또는 4장과 현재 선택 0~2장, 확정 여부, 남은 시간
- 덱 제거 후보와 현재 선택, 남은 시간
- 진행 중인 전투 재생 배치와 최종 서버 상태

성장 후보는 재접속 때 다시 생성하지 않습니다. 이미 확정한 선택은 변경할 수 없습니다.

## 채팅

- 같은 방 플레이어에게만 전송
- 서버 세션의 playerId/닉네임 사용
- 앞뒤 공백 제거, 연속 공백 정규화, 1~100자
- 플레이어당 초당 최대 2개
- 최근 50개를 보관하고 재접속 시 복원
- 사망자는 새 메시지를 보낼 수 없음
